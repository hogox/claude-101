Si Cowork te enganchó, este es el horizonte. **Claude Code** es la herramienta agéntica de Anthropic que vive en la **terminal** (línea de comandos): lee una base de código completa, edita archivos, ejecuta comandos y tests, y se integra con GitHub. Está pensada para **desarrolladores** y perfiles técnicos.

## Cowork y Claude Code: la misma potencia, dos públicos

Anthropic lo dice claro: *"Cowork es la misma capacidad de Claude Code con una experiencia simplificada"*. De hecho, Cowork nació porque equipos no técnicos empezaron a usar Claude Code para tareas que no eran programar.

| | **Cowork** | **Claude Code** |
|---|---|---|
| Dónde vive | App de escritorio | Terminal (y editores como VS Code) |
| Para quién | Trabajo de oficina, sin saber programar | Desarrolladores y curiosos técnicos |
| Qué hace | Archivos, informes, datos, tareas multipaso | Leer/editar código, automatizar, abrir PRs |

> 👉 Para tu trabajo de oficina, **Cowork es tu mejor puerta de entrada**. Claude Code es el paso siguiente solo si quieres meterte en automatización o desarrollo más técnico.

## ¿Tiene sentido para ti?

Si te mueves cómodo con archivos y carpetas pero nunca tocaste una terminal, quédate con Cowork: te da el 90% del valor sin la curva técnica. Si te pica la curiosidad por automatizar más (scripts, integraciones, tus propias herramientas), Claude Code es a dónde apuntar. Requiere un plan de pago (Pro/Max o asiento de equipo) y nociones básicas de terminal y de código.

📝 **Ejemplo — lo que hace Claude Code:** mapea y explica un proyecto entero en segundos, hace cambios que tocan muchos archivos a la vez, ejecuta los tests y abre un pull request con el resultado. Empresas lo usan para migraciones grandes y resolver incidentes mucho más rápido.

🎯 **Ejercicio (10 min):** no instales nada todavía. Solo mira el video de abajo o entra al curso oficial y decide si Claude Code es para ti **ahora** o un objetivo para más adelante. Dominar Cowork primero es la mejor preparación.
